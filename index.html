Privacy Policy for Simple Automation Runner

Last updated: May 29, 2026

Simple Automation Runner is a Chrome extension that allows users to create, save, and run browser automation scenarios configured by the user.

Data Collection and Storage

The extension may store the following data locally on the user's device using Chrome extension storage:

  • Automation scenario names
  • Start URLs configured by the user
  • CSS selectors selected by the user
  • Values entered by the user in automation steps
  • Keyboard inputs configured by the user
  • Recent run logs, including success or error status

This data is stored locally in the user's browser. Simple Automation Runner does not send this data to an external server.

Data Sharing

Simple Automation Runner does not sell, rent, trade, or share user data with third parties.

Advertising and Tracking

Simple Automation Runner does not inject advertisements, does not use user data for advertising, and does not track browsing behavior for marketing or analytics purposes.

Use of Website Data

When a user runs an automation scenario, the configured actions may be performed on the target website selected by the user. Any data entered into that website may be processed by the target website according to its own privacy policy.

Permissions

The extension uses Chrome permissions such as storage, tabs, scripting, activeTab, sidePanel, and host permissions only to provide user-configured browser automation functionality.
